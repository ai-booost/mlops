## 2. 커맨드



### Fire 커맨드

```bash
python weather_collect.py run_data_prepare --help 
```

### 파라미터 

```json
{
    "start_year": "시작년도",
    "start_month": "시작월",
    "month_range_count": "가져올 개월 수",
    "output_file": "출력파일명"
}
```

### 실행 예시

```bash
python weather_collect.py run_data_prepare --start_year=2023 --start_month=1 --end_year=2023 --end_month=2 --output_file="kma_data2.csv"
```
